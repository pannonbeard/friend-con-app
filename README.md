# Overview

This application is our way to keep track of our silly little convention from year to year.

## To Run
- Requires 
  - Ruby 3.2.2
  - Ruby on Rails
  - PostgreSQL

### Start up
``` 
bundle install
rails db:setup
rails server 
```
